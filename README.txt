Для запуска приложения:
-Перейдите в папку MO_hack в консоле (cd ../../MO_hack)
-Установите зависимости командой pip3 install -r requirements.txt
-Пропишите в консоле python app.py

Или

-Откройте папку через интерпретатор
-Установите зависимости командой pip3 install -r requirements.txt
-Запустите app.py
